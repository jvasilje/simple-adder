# Simple Adder

A simple Python package for adding two numbers.

## Installation

```bash
pip install simple-adder
```

## Usage

```python
from simple_adder import add

result = add(2, 3)
print(result)  # Output: 5
```

## Development

### Setup

```bash
pip install -e ".[dev]"
```

### Running Tests

```bash
pytest
```
